Alpha v0.0.4

- Added new tutorial tab for all important keywords.
- Added effect animations to Distance and Range change effects.
- Added feedback button into the options menu.
- Fixed major bug that caused the game to crash after some time running the game due to an asset memory leak.
- Fixed cards positioning when an Herobot is destroyed.
- Improved card details window behaviour. You can now tap and press on a card and move the finger around the screen without causing the window to close.
- Updated some enemy model sizes.
- Updated UI display generation system.
- Updated effects & stats window’s spawn behaviour.
